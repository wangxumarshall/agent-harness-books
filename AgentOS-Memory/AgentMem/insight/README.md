# fs-vector-graph

本目录用于沉淀 `filesystem like agent memory` 与 `vector/graph like agent memory` 的专题研究结果。

文件说明：

- `report.md`
  主报告。覆盖两大类的背景、技术方案、技术效果、优劣势、应用场景，以及对 Cortex-Mem 的借鉴。
- `evidence-matrix.md`
  逐系统证据矩阵。用于区分强证据、弱证据与是否进入主线结论。
- `sources.md`
  官方来源索引。汇总本轮研究使用的官方仓库、文档、论文和站点。

阅读顺序建议：

1. 先看 `report.md`
2. 再用 `evidence-matrix.md` 检查证据强弱
3. 最后在 `sources.md` 中追溯原始资料
